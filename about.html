Hi!

My name is Evan Stoddard. I'm an embedded software engineer living in Nashville, TN originally from Buffalo, NY (go Bills!)

I currently work for Skip Line, LLC working on making roadways safer as well as trying to modernize infrastructure.

This site is a collection of various projects I'm working on. Some technical, some not. Please enjoy and if anything sparks interest, please feel free to reach out!
